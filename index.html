// how it works

Three moves, memorize zero.

01

Summon

Press ⌥ space or click the floating circle. Lume grabs one frame of your screen.

02

It speaks first

Claude opens the conversation: what it sees, the likely blocker, and an offer to help. Never an empty chat box.

03

Choose your depth

Explain it, get the steps, be walked through with highlights — or hand over the mouse with Autopilot.

// bring your own claude

Two ways to power it.

Option A · API key

Anthropic API

Grab a key at console.anthropic.com, paste it into Lume's Settings (menu bar → Settings). Fastest replies; pay-per-use on your Anthropic account. Stored only in your Keychain.

Option B · Subscription

Claude Pro / Max

Already pay for Claude? Install the Claude Code CLI and log in once:

npm i -g @anthropic-ai/claude-code → run claude

Lume auto-detects it — no API billing, replies a touch slower.

// privacy

Built paranoid, on purpose.

◉

Capture only on ask. A click, a hotkey, a message, or an Autopilot step you initiated. Never on a timer, never in the background.

↯

Nothing stored. Screenshots go to Anthropic's Claude and are discarded after each reply. No disk writes in API mode.

⌗

Keys in Keychain. Your API key never touches a file, a default, or a log.

■

Hard pause. One menu-bar click disables all capture and greys the circle until you say otherwise.
